# PET lid style photos

The "Lid styles" section on `/pet-cups` shows ONE representative photo per lid
type (deliberately not per-SKU — per-size availability lives in the rate
sheet). The three files it loads:

| File              | Style |
|-------------------|-------|
| `type-flat.jpg`   | Flat lid (straw slot) |
| `type-dome.jpg`   | Dome lid (straw slot) |
| `type-sipper.jpg` | Sipper / strawless lid |

To replace a photo, overwrite the file (square, ~640px, product centred).
Source: Jingyu 98mm rate-request sheet, Jun 2026.
